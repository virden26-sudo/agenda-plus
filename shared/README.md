Kotlin Multiplatform (KMM) shared module for Agenda_plus

Overview
- Contains shared data models and JSON utils used by Android and (when built on macOS) iOS.

Build & usage
- Android (Windows/Linux/macOS): build as part of the Android app normally with Gradle.
- iOS (macOS required): enable CocoaPods integration and build the framework for Xcode.

On macOS, from the project root run:

```bash
./gradlew :shared:podInstall
./gradlew :shared:assemble
```

Then open the Xcode workspace generated by CocoaPods (or integrate the produced framework from `shared/build` into your Xcode project).

Notes
- This module conditionally configures iOS targets only when running on macOS to avoid Kotlin/Native toolchain downloads on non-macOS environments.
- Implement platform-specific expect/actual code in `commonMain`/`iosMain` as needed (networking, image handling, platform APIs).
